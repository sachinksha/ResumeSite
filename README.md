# Project Outline
Website to showcase a given person's profile in a resume/CV format.
AI reader friendly website as well as human friendly website.

The input to the site is from an excel file, so any new file can also be uploaded to get a nice view.

Sites settings view should download/import/export have options: 
1. download excel template
1. upload a new excel file of the resume, 
1. download a pdf of the raw (all the data), 
1. prominent and most important feature of the site - download the pdf of the resume with the current config/settings to show/hide collapse/expand specific views and fields.
PDF export is based on standard A4 size page. Condensed neat appearance for a professional CV that is AI reader friendly and human friendly too while being printer friendly.

Main/home page view of the site must show the professional resume view with the config/settings applied to the data.
Site must support light and dark theme.
   
All the resume fields are to be taken up as the input via the excel file and config/settings view shall have options to collapse or expand specific fields.
Some of the fields shall have concise or detailed view. So a summary text and a detailed text version shall need to be saved against such fields.
Config or settings for what fields to view or hide, which fields to show how should also be saved to the excel.

Some fields may be kept as private or hidden especially date of birth / age, side projects, other interests, etc
Fields like address can be detailed or a summary.
Cards for work experience can show few or all fields.
Typically work experience cards are ordered from most recent to the oldest.
Side projects / interests, Other interests, achievements, languages may be listed with names or name and description
