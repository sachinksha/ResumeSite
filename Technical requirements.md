# Technical requirements
## Main tech stack and guidelines
Vite, Vue, Typescript

## UI and CSS
Responsive design
Light and dark theme support
No inline CSS. All styles have to be declared and applied properly.
Include a theme css file which can easily be replaced to use another color scheme throughout the app.

## Clean code
Excellent separation of concerns. Code needs to be clean and neatly organized into proper files and folders.
Design clean simple and small components that are then used in the larger components or views.
Code must be very easy to read and maintain.

## Auth and data
Support for Google based authentication and save data to google firebase.

## Deployment
Project will then be deployed live using vercel.

## Readme, technical documentation
Develop a neat readme file to study and learn the technical topics for a software developer - especially for learning vue, typescript and system design.
